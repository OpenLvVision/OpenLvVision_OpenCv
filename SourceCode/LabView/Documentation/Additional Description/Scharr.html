The function computes the first $x$- or $y$- spatial image derivative using the Scharr operator. The call:

{{< katex >}}$ \texttt{Scharr(src, dst, ddepth, dx, dy, scale, delta, borderType)} ${{< /katex >}}

is equivalent to:

{{< katex >}}$ \texttt{Sobel(src, dst, ddepth, dx, dy, FILTER\_SCHARR, scale, delta, borderType)} ${{< /katex >}}

The Scharr operator is typically more accurate than the Sobel operator for calculating gradients, especially for $3 \times 3$ kernels.
