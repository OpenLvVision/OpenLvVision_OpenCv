The function warpPerspective transforms the source image using the specified matrix:

{{< katex >}}$ \texttt{dst}(x,y) = \texttt{src}\!\left( \frac{M_{11}x + M_{12}y + M_{13}}{M_{31}x + M_{32}y + M_{33}},\; \frac{M_{21}x + M_{22}y + M_{23}}{M_{31}x + M_{32}y + M_{33}} \right) ${{< /katex >}}

This formula is used when the flag WARP_INVERSE_MAP is set. Otherwise, the transformation matrix is first inverted using invert and then applied.

The function cannot operate in-place.
